# Movie Recommendation System

## Overview

This project is an AI-powered Movie Recommendation System built using FastAPI, Streamlit, and Machine Learning techniques. The system recommends movies based on content similarity using TF-IDF vectorization and cosine similarity. It also integrates with the TMDB API to fetch movie details, posters, genres, and recommendations.

---

## Features

- Movie search functionality
- Content-based recommendation system
- TF-IDF vectorization
- Cosine similarity based recommendations
- TMDB API integration
- Genre-based recommendations
- Movie details and posters
- FastAPI backend
- REST API endpoints

---

## Tech Stack

### Backend
- Python
- FastAPI
- Pandas
- NumPy
- Scikit-learn
- HTTPX

### Frontend
- Streamlit

### Machine Learning
- TF-IDF Vectorizer
- Cosine Similarity

### API
- TMDB API

---

## Dataset Processing

The movie dataset was preprocessed using:
- Lowercasing
- Punctuation removal
- Stopword removal
- Lemmatization

TF-IDF vectorization was then applied to generate feature vectors for movie similarity calculation.
